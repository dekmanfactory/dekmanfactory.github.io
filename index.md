# 🏆 SQLD 자격시험 핵심요약 및 최종정리문제집

> SQL 개발자(SQLD) 자격증 취득을 위한 완벽한 학습 가이드

## 📋 블로그 소개

**SQLD(SQL 개발자) 자격시험**의 모든 출제 영역을 체계적으로 정리한 종합 학습 블로그입니다.
총 **150세트**의 이론 요약과 실전 문제를 통해 합격까지의 최단 경로를 제시합니다.

### 🎯 블로그 특징
- ✅ **핵심 이론 요약**: 각 주제별 필수 개념 정리
- ✅ **실전 문제**: 기출문제 유형 분석 및 예상문제
- ✅ **단계별 학습**: 기초 → 심화 → 실전 순차 학습
- ✅ **최신 출제경향**: 2024-2025 최신 기출 분석 반영

---

## 📚 학습 구성 체계

### 🔹 세트 구성
- **1세트** = 이론요약 1개 + 실습문제 3개
- **총 150세트** = 600개 파일 (이론 150개 + 문제 450개)
- **예상 학습기간**: 3-6개월 (1일 1세트 기준)

- **Set 11-20**: 엔터티 개념, 분류, 명명규칙, 관계 등

#### 🔸 3장: 속성 및 식별자 (10세트)
- **Set 21-30**: 속성 분류, 도메인, 식별자 유형 등

### 🏗️ 제2과목: 데이터베이스 구축 (40세트)

#### 🔸 1장: 논리 데이터베이스 설계 (15세트)

#### 🔸 2장: 물리 데이터베이스 설계 (15세트)
- **Set 46-60**: 인덱스, 파티셔닝, 클러스터링 등

#### 🔸 3장: SQL 기초 (10세트)
- **Set 61-70**: DDL, DML, DCL 기본 문법

### 💻 제3과목: SQL 기본 및 활용 (80세트)

#### 🔸 1장: SQL 기본 (30세트)

#### 🔸 2장: SQL 활용 (30세트)
- **Set 101-130**: 조인, 서브쿼리, 집합연산자, 윈도우함수 등

#### 🔸 3장: 관리 구문 (20세트)
- **Set 131-150**: 프로시저, 함수, 트리거, 사용자 관리 등
